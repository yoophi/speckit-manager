# speckit-manager

Electron desktop app scaffolded with `electron-vite`, using React, TypeScript, React Router, Tailwind CSS v4, and shadcn/ui.

## Repository Layout

The application source currently lives in:

```text
./speckit-manager
```

## Getting Started

```bash
cd speckit-manager
npm install
npm run dev
```

## Available Scripts

```bash
cd speckit-manager

# type checking
npm run typecheck

# development
npm run dev

# production builds
npm run build:mac
npm run build:win
npm run build:linux
```

## Frontend Stack

- React 19
- React Router with `HashRouter`
- Tailwind CSS v4 via `@tailwindcss/vite`
- shadcn/ui with the New York style preset
